1. Build against Google APIs 2.3.3. (API Level 10)
2. If building from source, make sure to sign with the debug keystore
   (Instructions available in misc/README, keystore in misc/debug.keystore,
   information in misc/keyinfo.txt)
3. Install APK, using Eclipse or ADT toolkit.


Limitations:
Only one set of directions and one map overlay are available (hardcoded).
